# Brute Force Login Detection Script

## Overview
This project demonstrates a Python-based security script that detects potential brute-force login attacks by analyzing authentication logs.

## Objective
Identify repeated failed login attempts from the same IP address.

## Tools Used
- Python
- Regex
- Linux authentication logs

## Features
- Reads log files
- Detects failed logins
- Flags suspicious IP addresses

## Skills Demonstrated
- Security log analysis
- Threat detection
- Python autom
